# Personal Portfolio Website

## Project Overview

This project is a simple personal portfolio website developed using HTML5. The objective of this project is to demonstrate fundamental HTML concepts such as webpage structure, semantic tags, navigation links, images, lists, and forms.

## Features

* About Section
* Skills Section
* Contact Form
* Navigation Menu
* Profile Image
* Semantic HTML Tags

## Technologies Used

* HTML5

## Project Structure


## Setup Instructions

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Use the navigation menu to move between sections.

## HTML Concepts Used

* HTML5 Document Structure
* Headings and Paragraphs
* Links and Navigation
* Images and Alt Text
* Forms and Input Validation
* Lists
* Semantic HTML Tags

## Portfolio Structure

### Header

Contains the website title and navigation menu.

### About Section

Displays personal information and profile image.

### Skills Section

Shows a list of skills using HTML list elements.

### Contact Section

Contains a contact form with Name, Email, and Message fields.

### Footer

Displays copyright information.

## Technical Details

### Architecture

* Single-page website
* Built using HTML5
* Uses semantic HTML structure

### Data Structures

* Unordered List (`<ul>`) used for displaying skills
* Form elements used for collecting user input

### Algorithms

No complex algorithms are used in this project. Internal page navigation is implemented using anchor links.
